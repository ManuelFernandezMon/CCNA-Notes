# 11.6 — Subnetear Prefijos /16 y /8

---

## 🏢 Subnetear un /16 — redes empresariales

> [!info] Mas bits de host para tomar prestados
> En un /16 hay 16 bits de host disponibles. Puedes tomar prestados hasta 14 (los ultimos 2 no se pueden prestar — quedarian 0 hosts).

```
  /16 original:
  nnnnnnnn.nnnnnnnn.hhhhhhhh.hhhhhhhh
  └──── RED 16 ────┘└──── HOST 16 ───┘
```

| Prefijo | Mascara | Subredes | Hosts/subred |
|---------|---------|----------|--------------|
| `/17` | 255.255.128.0 | 2 | 32,766 |
| `/18` | 255.255.192.0 | 4 | 16,382 |
| `/20` | 255.255.240.0 | 16 | 4,094 |
| `/22` | 255.255.252.0 | 64 | 1,022 |
| `/24` | 255.255.255.0 | 256 | 254 |
| `/26` | 255.255.255.192 | 1,024 | 62 |

---

## 📐 Ejemplo: crear 100 subredes en un /16

> [!example] Empresa con 172.16.0.0/16 que necesita 100 subredes
> ```
> Necesito: minimo 100 subredes
> 2^6 = 64  → insuficiente
> 2^7 = 128 → ¡suficiente! (cubre las 100)
>
> Tomo prestados 7 bits → /16 + 7 = /23
> Resultado: 128 subredes de 510 hosts cada una
> ```

---

## 🌐 Subnetear un /8 — ISPs y redes enormes

> [!info] 24 bits de host disponibles
> En un /8 puedes tomar prestados hasta 22 bits para subnetear.

> [!example] ISP con 10.0.0.0/8 que necesita 1000 subredes
> ```
> Necesito: minimo 1000 subredes
> 2^9  = 512  → insuficiente
> 2^10 = 1024 → ¡suficiente!
>
> Tomo prestados 10 bits → /8 + 10 = /18
> Resultado: 1024 subredes
> ```

---

## 🔢 Potencias de 2 (memorizar para el examen)

```
  2^1 = 2        2^7  = 128
  2^2 = 4        2^8  = 256
  2^3 = 8        2^9  = 512
  2^4 = 16       2^10 = 1,024
  2^5 = 32       2^11 = 2,048
  2^6 = 64       2^12 = 4,096
```

> [!tip] 💡 El metodo universal
> 1. ¿Cuantas subredes necesito? → busca la potencia de 2 que la cubra → esos son los bits a prestar
> 2. ¿Cuantos hosts por subred? → 2^(bits host restantes) − 2
> 3. Suma los bits prestados al prefijo original
